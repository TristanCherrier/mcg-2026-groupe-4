# MCG 2026 - Groupe 4

Projet Godot PC base sur le projet d'origine du groupe.

Le socle conserve vos scenes et scripts de base :
- `main.tscn`
- `classroom.tscn`
- `robot.tscn`
- `Tiny Car.tscn`
- `ship.tscn`

Contenu du rendu :
- Niveau 1 : parcours dans la salle UBE avec 3 enigmes scientifiques, monstres et sortie debloquee.
- Niveau 2 : parcours en vehicule avec obstacles fixes et mobiles, projectiles, checkpoints et integrite.
- Niveau 3 : deplacement au propulseur oriente, collecte de 5 anneaux, consommation et recharge d'energie, puis plateforme finale.
- Interface 2D : ecran titre, menu principal, HUD, pause, victoire, defaite et score global.

Commandes principales :
- Robot : `ZQSD`, souris, `E` pour interagir, `V` pour changer de vue.
- Voiture : `Z` accelerer, `S` freiner, `Q/D` diriger, `Espace` frein a main.
- Propulseur : `Z` pousser, souris pour orienter le reacteur.

Blender :
